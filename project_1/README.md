# Project 1 - Qt Hello Command Line

A simple Qt-based command-line application that prints "hello".

## Prerequisites

- Qt 6 (or Qt 5 with minor modifications)
- CMake 3.16 or higher
- C++17 compatible compiler

## Building

```bash
mkdir build
cd build
cmake ..
cmake --build .
```

## Running

```bash
./project_1
```

## Output

```
hello
```
